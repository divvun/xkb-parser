# X11 Keyboard file parser

Parses `.xkb` (X keyboard extension) files.
Especially the ones created by [kbdgen].

[kbdgen]: https://github.com/divvun/kbdgen/blob/c5f44e557fe6a8b56ca244747f84ba369b3a5502/kbdgen/gen/x11.py
